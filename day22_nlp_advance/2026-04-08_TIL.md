## 1. Naive Bayes 기반 감정 분석
베이즈 정리를 활용해 텍스트가 특정 카테고리에 속할 확률을 계산하는 모델을 실습했다.

### 💻 핵심 실습 코드

```python
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.naive_bayes import MultinomialNB

# 1. 텍스트 데이터의 수치화 (Vectorization)
cv = CountVectorizer()
train_doc = ["i am happy", "this is great", "i am sad", "this is bad"]
labels = ["positive", "positive", "negative", "negative"]
train_matrix = cv.fit_transform(train_doc)

# 2. 나이브 베이즈 모델 학습
clf = MultinomialNB()
clf.fit(train_matrix, labels)

# 3. 예측 (Smoothing과 Log 확률이 내부적으로 적용됨)
test_doc = ["i feel great"]
test_matrix = cv.transform(test_doc)
print(f"Prediction: {clf.predict(test_matrix)}") # 결과: ['positive']
```

## 2. 한국어 형태소 분석 (KoNLPy)
한국어의 형태적 특성(교착어)을 처리하기 위해 다양한 형태소 분석기를 비교 실습했다.

### 💻 핵심 실습 코드

```python
from konlpy.tag import Kkma, Okt

sent = "자연어 처리는 정말 재밌어! 너의 이름은 뭐야?"

# Kkma: 분석이 정교하며 문장 분리 기능 제공
kkma = Kkma()
print("Kkma Nouns:", kkma.nouns(sent))

# Okt: 속도가 빠르고 스테밍(어간 추출) 지원
okt = Okt()
print("Okt POS (Stemming):", okt.pos(sent, stem=True))
```

## 3. 문장 및 문서 유사도 (Similarity)
문장을 수치화(Vectorization)하여 수학적 거리를 계산하는 기법을 익혔다.

### 💻 핵심 실습 코드 (Cosine Similarity)

```python
import numpy as np

def cal_cosine_sim(v1, v2):
    # 두 벡터 사이의 각도를 이용한 유사도 측정 (0~1)
    dot_product = np.dot(v1, v2)
    l2_norm = (np.sqrt(np.dot(v1, v1)) * np.sqrt(np.dot(v2, v2)))
    return dot_product / l2_norm

# 예시 벡터 비교
v_a = np.array([0.1, 0.5, 0.8])
v_b = np.array([0.2, 0.4, 0.9])
print(f"Cosine Similarity: {cal_cosine_sim(v_a, v_b):.4f}")
```

## 🛠️ Trouble Shooting & Debugging (오늘의 삽질 기록)

### 1) AttributeError: module 'numpy' has no attribute 'object'
- 원인: NumPy 1.24 버전부터 np.object가 삭제되어 TensorFlow와 충돌 발생.
- 해결: 라이브러리 버전을 1.24 미만으로 다운그레이드하여 호환성 확보.
- 조치: pip install "numpy<1.24"

### 2) UnicodeDecodeError (cp949)
- 원인: 윈도우 환경에서 UTF-8 텍스트 파일을 읽을 때 발생하는 인코딩 오류.
- 해결: 파일 오픈 시 encoding='utf-8' 옵션을 명시적으로 추가.
