변수와 자료형

Python은 값을 변수에 저장하고, 그 값을 조합하면서 프로그램 흐름을 만듭니다.

짧게 다시 정리
핵심Python은 문법보다 흐름 제어 감각이 중요합니다. 값을 저장하고, 조건을 나누고, 반복하고, 기능을 묶는 방식이 핵심입니다.
연결머신러닝, 딥러닝, API, 파일 처리, 프로젝트까지 결국 Python 기초가 바탕이 됩니다.
