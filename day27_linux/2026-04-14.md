# Day27

## 오늘 배운 내용

오늘은 리눅스와 우분투 기본 환경에 대해 학습했다.  
리눅스의 커널과 쉘 구조를 이해하고, GNU 프로젝트와 오픈소스 생태계가 어떤 흐름으로 이어지는지도 정리했다.

우분투 환경에서는 `apt`를 활용한 패키지 관리 방식을 배웠고,  
`ls -al`, `cd`, `chmod`, `sudo` 같은 기본 명령어를 직접 실습하면서 CLI 환경에 조금 더 익숙해졌다.

또한 프로세스 제어도 함께 실습했다.  
`Ctrl + C`는 실행 중인 작업을 종료하고, `Ctrl + Z`는 작업을 일시 정지한다는 차이를 배웠다.  
정지된 작업은 `fg` 명령어로 다시 실행할 수 있다는 점도 확인했다.

텍스트 편집기인 Nano와 Vim도 사용해봤다.  
특히 Vim에서 저장 후 종료하는 `:wq`, 저장하지 않고 나오는 `:qa!` 명령어를 연습하면서 기본 조작법을 익혔다.

마지막으로 KoGPT-2 모델을 활용한 AI 챗봇 실습을 진행했다.  
단순히 모델을 불러오는 것에서 끝나는 것이 아니라, 키워드 매핑을 통해 '헬피'라는 페르소나를 반영하는 로직도 구현했다.  
이를 통해 AI 모델 출력에 원하는 성격과 응답 스타일을 입히는 과정을 직접 경험할 수 있었다.

## 느낀 점

리눅스는 처음에는 낯설었지만, 직접 명령어를 입력해보면서 GUI 없이도 시스템을 다룰 수 있다는 점이 재미있게 느껴졌다.  
또한 챗봇 실습에서는 단순 모델 실행보다, 내가 원하는 캐릭터를 반영하도록 로직을 설계하는 과정이 더 인상적이었다.

## 어려웠던 점

- 아나콘다 풀버전 설치 과정에서 용량과 시간이 많이 소요됐다.
- 윈도우 환경과 WSL2 리눅스 환경이 분리되어 있어 설치 위치를 헷갈렸다.
- 환경을 나눠서 생각하는 습관이 아직 더 필요하다.
