# 👨‍🍳 냉장고 파머 Pro (FridgeCook Pro)

내 냉장고 속 재료를 선택하면, 보유 재료와의 일치도를 기반으로 최적의 레시피를 추천해주는 스마트 웹 서비스입니다.

---

## 📌 About

이 프로젝트는 사용자가 냉장고에 있는 재료를 직접 입력하거나 선택하면,  
SQLite에 저장된 레시피 데이터와 비교하여 가장 적합한 요리를 추천해주는 Flask 기반 웹 애플리케이션입니다.

공공데이터포털 레시피 API 데이터를 활용했으며,  
비정형 재료 텍스트를 정제하고 조리 단계를 구조화하여 실제로 사용할 수 있는 형태의 서비스로 구현했습니다.

---

## ✨ 주요 기능

### 1. 재료 기반 레시피 추천
- 사용자가 입력하거나 선택한 재료를 기준으로 레시피를 추천
- 보유 재료와 레시피 재료의 일치 비율을 점수화하여 상위 결과 제공

### 2. 데이터 정제
- 공공데이터 API의 재료 텍스트에서 단위, 괄호, 불필요한 표현 등을 정규표현식으로 정제
- 추천 로직에 활용 가능한 형태로 전처리

### 3. 카테고리형 재료 사이드바
- 수천 개의 재료를 고기, 수산물, 채소, 유제품, 양념 등으로 분류
- 아코디언 UI를 통해 원하는 재료를 쉽게 탐색 가능

### 4. 상세 조리 단계 제공
- 최대 20단계의 조리법을 불러와 출력 가능
- 단계별 설명과 이미지 데이터를 함께 제공할 수 있도록 설계

### 5. 요리 친화적 UX
- 요리 중에도 확인이 쉽도록 조리 단계 중심의 사용 흐름을 고려해 구현

## 🔍 Core Logic

1. SQLite 데이터베이스에서 레시피 정보를 불러옵니다.
2. 재료 문자열에서 줄바꿈, 괄호, 단위 등을 제거하여 정제합니다.
3. 사용자가 선택한 재료와 각 레시피 재료를 비교합니다.
4. 일치하는 재료 수를 기반으로 점수를 계산합니다.
5. 점수가 높은 순으로 상위 12개의 레시피를 추천합니다.

## 🚧 Limitations / Improvements

- 현재는 부분 문자열 매칭 방식이라 유사 재료 간 오매칭 가능성이 있음
- 추천 점수가 단순 비율 기반이라 핵심 재료 반영이 부족할 수 있음
- 재료 동의어 처리 및 정확도 개선이 가능함
- 추후 로그인, 즐겨찾기, 최근 본 레시피 기능 등으로 확장 가능
